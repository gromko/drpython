Є кілька речей, які вам потрібно знати про те, як працюють налаштування.

По-перше, це формат файлу. Налаштування зберігаються у текстовому форматі розмітки. Це означає, що якщо в майбутніх версіях будуть додані нові налаштування, вам не потрібно буде змінювати формат або запускати якусь утиліту для файлу налаштувань.

Налаштування

Імпорт/Експорт

У нижній частині загальної панелі є невелика кнопка під назвою "Налаштування". Вона запускає діалог, який дозволяє вам імпортувати/експортувати:

  • налаштування, скорочення, контекстне меню, панель інструментів;
  • плагіни;
  • drscripts;
  • все вищезазначене.

Це полегшує налаштування певного середовища. (Наприклад, ви можете поширювати SillyDefaults.zip, який включає налаштування, плагіни, та drscripts, які ви хочете, щоб усі отримувачі встановили.)

(Ви також можете використовувати функції, визначені в drZip.py, щоб автоматизувати цей процес за допомогою простого скрипту Python).

ПОПЕРЕДЖЕННЯ: Імпорт перезапише дані!

DrPython перезапише будь-які дублікати під час імпорту. Це означає:

  • Налаштування будуть перезаписані.
  • Також скорочення, контекстне меню та панель інструментів.
  • Які плагіни завантажуються при запуску, і будь-які дублікати будуть перезаписані.
  • Меню drscript буде перезаписано.
  • Також будь-які дублікати будуть перезаписані.

Після імпорту вам потрібно перезапустити DrPython, щоб побачити зміни.

Під час експорту DrPython зберігатиме drscripts у каталозі налаштувань (у zip-файлі). Структура каталогу буде відображати меню. Це означає, що коли ви потім імпортуєте drscripts, вони будуть фізично розташовані у вашому каталозі налаштувань, у підкаталозі 'drscripts'.

Документ

Line Number Margin Width
Встановлює ширину поля для номерів рядків у пікселях.
Autoindent
Вибір методу автоматичного відступу:
None:
Залишити як є!
Normal:
Додає кількість табуляцій до першого непробільного символу в попередньому рядку при кожному новому рядку.
Context Sensitive:
Те саме, що й Normal, але додає додатковий відступ після ':', і зменшує відступ після наступних операторів: return, break, pass, continue, raise.

Автоматично перевіряє (при активації DrPython), чи були змінені будь-які відкриті файли зовнішнім джерелом. Потім пропонує користувачу перезавантажити файл. Якщо ні, час модифікації для документа (у програмі) скидається.

Якщо увімкнено, використовує тип відступу завантаженого файлу. (Працює лише при відкритті). Іншими словами: якщо тип відступу за замовчуванням — це табуляція, але ви завантажуєте файл, який використовує пробіли, DrPython буде використовувати пробіли для відступу для цього документа, продовжуючи використовувати табуляцію за замовчуванням для інших документів.
Use Tabs
Якщо позначено, використовує табуляцію '\t' при натисканні клавіші табуляції, і для автовідступу. Якщо ні, використовує кількість пробілів, вказану у ширині табуляції.

Не показує горизонтальну смугу прокрутки і "переносить" довгі рядки на кілька рядків.

Use Indentation Guides
Додає вертикальні лінії, які візуально з'єднують рівні відступу.

Встановлює стовпець, який вказує, що ви вийшли за певну позицію (наприклад, 79). Використання від'ємного числа вказує DrPython виділяти текст після вказаної позиції, замість малювання стовпця. 0 вказує DrPython нічого не малювати.
Scroll Extra Page
Вказує DrPython встановити діапазон вертикальної прокрутки на одну сторінку більше, ніж довжина поточного документа.
Highlight Current Line
Виділяє поточний рядок, використовуючи фон із стилю курсора.
Comment Mode
Встановлює, де DrPython розміщує рядок коментаря. Якщо "Початок рядка", то на початку кожного рядка у виділеній області. Якщо "Початок слова", то на початку першого слова кожного рядка у виділеній області.

Підсвічує текст, який не розпізнається через розширення.
Only Use Default Syntax Highlighting
Використовує лише підсвічування синтаксису, вказане вище, незалежно від розширення файлу.
Style
Встановлює, як виглядають різні типи тексту (колір тексту та фону, шрифт, розмір, підкреслення/жирний/курсив). Ви можете вибрати не використовувати стилі взагалі, використовувати їх або використовувати лише "Звичайний" та "Колір курсора".

Діалог файлів

Це встановлює поведінку діалогів файлів у DrPython.

Use wx.FileDialog
Якщо ви хочете використовувати стандартний діалог файлів wxPython, позначте цю опцію. У цьому випадку буде використовуватися лише налаштування маски файлів.
Wildcard
Це рядок, який використовується wxPython для визначення типів файлів для відображення. Формат: ОписA|Шаблон розширенняA1;Шаблон розширенняA2 ОписB|Шаблон розширенняB1;Шаблон розширенняB2

'|' розділяє типи файлів. Ось приклад:
Файли Python (*.py *.pyw)|*.py;*.pyw|Всі файли (*)|*

Ви можете використовувати редактор для додавання записів до рядка маски.

Якщо всі записи видалені, автоматично встановлюється Всі файли (*)|*.

Constant Wildcard
Ви можете встановити це в редакторі маски. Це просто шаблон (наприклад, *.py), який завжди видимий, незалежно від типу файлу, обраного у діалозі файлів.
Windows Shortcut Replace Table
Це потрібно/використовується лише на не-Windows системах.

Це містить інформацію для заміни рядків, знайдених у ярликах Windows (*.lnk), на правильні шляхи. Наприклад, якщо ваш диск C: змонтований на /mnt/win_c, ви можете ввести "C" для шаблону Windows та "/mnt/win_c" для рядка заміни.

Таблиця заміни використовує регулярні вирази для шаблону Windows та два спеціальні символи для рядка заміни. Отже, запис:

Шаблон Windows: "[A-D]"
Рядок заміни: "/mnt/win/@"

замінить будь-який диск A-D (наприклад, C: на /mnt/win/c) з рядком у нижньому регістрі. Якщо рядок заміни був /mnt/win_&, він дасть /mnt/win_C.

Будь-який дійсний регулярний вираз працюватиме для шаблону Windows.

Це розширення файлів, які розпізнаються для кожного типу файлу. Розділяються комами (наприклад, 'py,pyw,c,h').

Якщо позначено, використовує табуляцію \t при натисканні клавіші табуляції, і для автовідступу. Якщо ні, використовує кількість пробілів, вказану у ширині табуляції.

Не показує горизонтальну смугу прокрутки і "переносить" довгі рядки на кілька рядків.

За замовчуванням UNIX (\n). Ви також можете вибрати Windows (\r\n) або Mac (\r).

Пошук/Заміна

Більшість із цих налаштувань є стандартними для діалогів пошуку/заміни/переключення. Останнє регулює поведінку діалогу пошуку/заміни.

Regular Expression
Оцінює текст пошуку як регулярний вираз. Шукає лише вперед.
Match Case
Чутливий до регістру пошук.
Find Backwards
Пошук з кінця цільового тексту.
Whole Word
Шукає лише цілі слова (пробіли з обох сторін).
In Selection
Якщо є виділений текст, то пошук лише у виділеному тексті.
From Cursor
Пошук з поточної позиції у документі, а не з початку.
Prompt on Replace
Запитує користувача при кожній операції заміни, дозволяючи вирішити, чи замінити цільовий текст, чи пропустити його.
Auto Wrap
Якщо увімкнено, коли під час пошуку досягається початок/кінець документа, автоматично переходить на початок.
Under Cursor
Діалог пошуку/заміни встановлює слово під курсором як текст пошуку, якщо текст не виділено.

Це кодування, яке використовується за замовчуванням.  (Якщо автоматичне визначення увімкнено, drpython спробує спочатку unicode спочатку).
Ви можете вручну ввести кодування, залишити порожнім для вимкнення або вибрати зі списку кодувань:
<локаль за замовчуванням (якщо доступна), ascii, latin-1, utf-8, та utf-16.

Обмежує перевірку синтаксису при збереженні лише вказаними розширеннями. Формат: лише розширення (без крапки), розділені пробілами (наприклад): py pyw python буде виконувати перевірку синтаксису при збереженні (якщо вона увімкнена) лише для файлів з розширеннями py, pyw, або python.
Prompt On Save All
Запитує перед збереженням усіх змінених документів.
Max Number of Recent Files
Вказує DrPython, скільки останніх файлів слід запам'ятовувати. 0 — це мінімум, і це призведе до того, що DrPython ігноруватиме останні файли.
View Line Endings With Whitespace
Показує символи кінця рядка, коли пробіли видимі.
Check Line Ending Format on Open
Вказує DrPython перевіряти, чи використовується ваш тип кінця рядка за замовчуванням при відкритті файлу.
Check Indentation Type on Open
Вказує DrPython перевіряти, чи використовується ваш тип відступу за замовчуванням у відкритому файлі.
Python Arguments
Надсилає вказані аргументи безпосередньо до Python при запуску файлу, налагодженні файлу або запуску інтерпретатора Python окремо. Поширений аргумент — -i. Це вказує Python перейти в інтерактивний режим після завершення поточного файлу. Більше аргументів можна знайти у документації Python або ввівши python -h у командному рядку.
Auto-Detect UTF-8 Encoding
Автоматично визначає, чи файл має кодування UTF-8 (unicode).
Icon Size
Вказує DrPython, наскільки великою має бути панель інструментів. Вибір 0 (після перезапуску) вкаже DrPython не завантажувати панель інструментів взагалі.
Icon File
Якщо залишено порожнім або вказано недійсне ім'я файлу, DrPython завантажить стандартні файли. В іншому випадку, DrPython прочитає файл .dis та завантажить вказані в ньому зображення.
Докладніше про файли .dis
Default Directory
Вказує DrPython, де шукати файли спочатку. Після відкриття файлу DrPython встановлює поточний каталог на каталог файлу.
Enable Feedback
Дозволяє спливаючі вікна повідомляти про успішність операцій. (Наприклад, успішне оновлення налаштувань).
Always Prompt On Exit
Вказує DrPython завжди запитувати, чи користувач хоче вийти, навіть якщо немає поточного файлу або поточний файл збережено.
Backup File On Save
Кожного разу, коли ви зберігаєте файл, DrPython створює копію (ім'я файлу з доданим .bak) у тому ж каталозі перед записом у файл.

Якщо True, вказує DrPython показувати підказку за замовчуванням для кожного документа.
Use Tabs
Якщо позначено, використовує табуляцію \t при натисканні клавіші табуляції. Якщо ні, використовує кількість пробілів, вказану у ширині табуляції.
Tab Width
Вказує DrPython, наскільки великими виглядатимуть табуляції (\t). Наприклад, 8 зробить одну табуляцію схожою на 8 пробілів.
Line Ending Format
За замовчуванням UNIX (\n). Ви також можете вибрати Windows (\r\n) або Mac (\r).
Word Wrap
Не показує горизонтальну смугу прокрутки і "переносить" довгі рядки на кілька рядків.
Whitespace is Visible on Startup
Коли DrPython запускається, автоматично показує пробіли.
Scroll Extra Page
Вказує DrPython встановити діапазон вертикальної прокрутки на одну сторінку більше, ніж довжина підказки.
Vertical Size
Встановлює процентний розмір підказки відносно документа, коли підказка видима. 25% для підказки означає 75% для документа.
Style:
Встановлює, як виглядають різні типи тексту (колір тексту та фону, шрифт, розмір, підкреслення/жирний/курсив).

Ви можете вибрати не використовувати стилі взагалі, використовувати їх або використовувати лише "Звичайний" та "Колір курсора".

Браузер коду

Це вказує DrPython, як виглядає та працює браузер коду.

Position
Вказує, де з'являтиметься браузер коду: ліворуч, праворуч, зверху або знизу від поточного документа.
Visible By Default
Якщо True, вказує DrPython показувати браузер коду за замовчуванням для кожного документа.
Close On Activate
Коли ви активуєте елемент (вибираєте його подвійним клацанням або за допомогою клавіатури), це вказує DrPython, чи закривати вікно браузера коду.
Sorted(Alphabetical)
Вказує DrPython, чи сортувати класи/функції/імпортовані оператори за алфавітом, чи залишати їх у порядку, в якому вони з'являються у програмі.
Style
Встановлює, як виглядає браузер коду (колір тексту та фону, шрифт, розмір, підкреслення/жирний/курсив).
Auto Refresh on Save
Автоматично оновлює браузер коду при збереженні.
