Що таке DrPython?

Для початку, DrPython — це середовище розробки. По суті, це текстовий редактор з вбудованими інструментами програмування, такими як інтегрований командний рядок, маніпуляції з текстом та підсвітка синтаксису, щоб назвати кілька. Інтегрований командний рядок означає, що коли ви запускаєте програму на Python, DrPython перехоплює вивід, приймає ввід і в цілому поводиться як вікно консолі (наприклад, cmd.exe або rxvt).

DrPython був розроблений як крос-платформне середовище розробки Python. Основна мета полягала у використанні в освіті, щоб полегшити викладання програмування на Python. Спочатку він був розроблений на основі DrScheme. DrPython має грати ту саму роль для Python, яку DrScheme грає для Scheme.

Платформа розробки — Linux (наразі Mandrake 10.1). Він тестувався на різних дистрибутивах Linux (наприклад, Mandrake, Debian, Gentoo, Fedora), Windows 9x, Windows NT, Windows XP та Mac OS X.

Будь-яка платформа, яка підтримує Python 2.3+ та wxPython 2.5.2+, повинна працювати.

DrPython безкоштовний?

DrPython є безкоштовним та відкритим програмним забезпеченням. Це означає, що ви можете завантажити DrPython безкоштовно, ділитися ним скільки завгодно, переглядати вихідний код тощо. Ви навіть можете брати частини коду та вставляти їх у свою власну програму (за умови надання належного кредиту!). Ви навіть можете створити свою власну гілку DrPython або переписати код для роботи з іншою мовою.

Мета вільного/відкритого програмного забезпечення полягає в тому, що користувач має право використовувати своє програмне забезпечення так, як він цього бажає.

Недоліком цього є те, що DrPython не має гарантії. Тому, якщо після використання DrPython ваш комп'ютер стане телекінетичним і підірве ваш холодильник... Ви можете надіслати мені звіт про помилку.

Встановлення

Якщо ви читаєте це, DrPython вже має бути встановлений. Тим не менш, для роботи DrPython вам потрібні такі програми: Python 2.0 або новішої версії (тестувався з 2.2) та відповідна версія wxPython. Найкраще отримати останню версію Python та wxPython.

Якщо у вас виникають проблеми з запуском DrPython на Linux/Unix, запустіть файл drpython.lin, або відредагуйте перший рядок drpython.pyw або drpython.py, щоб він містив: 

#![ШЛЯХ/ДО/PYTHON]/python

(наприклад: #!/usr/bin/python). (Іншими словами, зробіть шлях абсолютним.)

Якщо у вас все ще виникають проблеми, перейдіть на сайт DrPython (http://www.sourceforge.net/projects/drpython/) та опублікуйте повідомлення на форумі допомоги.

Ви можете вручну встановити шлях для зберігання налаштувань:

drpython.pyw --preferencesbasepath="шлях"

DrPython буде використовувати налаштування в каталозі:
"шлях/.drpython"
("шлях/drpython" на Windows)

Закладки

Закладки досить прості: є закладки та папки закладок.

Закладка — це файл або каталог. Коли вона вибирається з меню, якщо це файл, він відкривається. Якщо це каталог, з'являється діалогове вікно відкриття в цьому каталозі.

Папка закладок може містити закладки та/або папки закладок.

Ви можете додати лише папку закладок або закладку до папки закладок.

Щоб перемістити закладку або папку закладок (і весь її вміст):

  1. Клацніть один раз на елемент, який хочете перемістити.
  2. Перетягніть його на елемент, після якого він має бути розташований.
  3. Перетягнувши його на будь-який елемент у папці, він буде доданий безпосередньо після цього елемента.
  4. Перетягнувши його на папку, він стане першим елементом у цій папці.

Перевірка синтаксису

Виконує вбудовану в Python функцію 'compile', а потім tabnanny, для поточного файлу. Повідомляє про синтаксичні помилки та помилки відступів.

Зауважте, що це перевіряє лише синтаксис поточного файлу.

Пошук/Заміна

Це досить стандартний/простий діалог. Однак є кілька речей, які варто відзначити.

Клацніть правою кнопкою миші на текстовому полі, щоб відкрити контекстне меню. Ви можете вставити табуляцію/новий рядок/повернення каретки, виконати операції з буфером обміну або очистити текст.

Ви також можете отримати доступ до історії пошуку/заміни через комбіноване поле.

Пошук та завершення

Це версія автозавершення, ініційована користувачем. DrPython буде шукати у поточному файлі можливі завершення слова, яке ви зараз вводите. Він відображатиме ці результати у невеликому вікні. Переміщайтеся за допомогою стрілок вгору та вниз, або для швидшого переміщення використовуйте page up та page down, або home та end, щоб перейти до початку та кінця списку.

Натискання Enter або Tab завершить поточне слово вибраним варіантом.

Контекстно-залежний: Пошук та завершення зупиняється на наступних символах (окрім пробілів): 

  ( ) < > [ ] { }

Це означає 

  tree
  dog(treat)

  dogs
  dog(tre

пошук на dog(tre дасть tree, treat. Однак: 

  tree
  dog(treat

  dogs

  dog(

пошук на dog( дасть dog(treat, dogs.

Примітка: Пошук та завершення чутливе до регістру.

Перехід до блоку / класу / функції

Кілька приміток. У коді: 

if True:
    print "Hello"
# Коментар
    print "Hello again"

'if True:' — це початок блоку;
'print "Hello again"' — це кінець.

Коментарі '#' ігноруються. Для явно (з'єднаних зі зворотним слешем) та неявно з'єднаних рядків, кожен рядок після першого ігнорується.

Коли зустрічається порожній рядок, він пропускається. Тому, якщо ви виберете 'Перейти до початку класу' з порожнього рядка, DrPython перевірятиме, доки не знайде рядок із кодом або кінець файлу, і почне звідти. (Спочатку перевіряється наступний рядок, потім попередній, далі наступний + 1, попередній - 1 тощо). Якщо клас або функція не знайдені, DrPython поводитиме себе так, ніби ви використали команду переходу до блоку.

Перехід до Traceback

Подвійне клацання на записі traceback у підказці (жирний рядок у прикладі нижче): 

Traceback (most recent call last):
 File "/home/user/Desktop/test.py", line 6, in ? wd NameError: name 'wd' is not defined

змусить DrPython відкрити '/home/user/Desktop/test.py' і перейти до рядка 6.

Відступи та табуляції

Опція "Використовувати табуляції" у розділі "Документ" у налаштуваннях вказує DrPython чи використовувати табуляції (за замовчуванням) чи пробіли для автоматичних відступів. "Ширина табуляції" у розділі "Загальні" визначає, скільки пробілів DrPython вважатиме за "Табуляцію".

Щоб побачити, які символи використовуються для відступів у поточному документі, перевірте рядок стану.

У меню "Пробіли" є пункти, які дозволяють встановити відступи на табуляції чи пробіли. Наприклад, встановлення відступів для документа на табуляції замінить певну кількість пробілів на табуляції. Ця операція виконується на початку кожного рядка.

Деякі програмісти вважають за краще (або їх навчили) використовувати табуляції ('\t') для відступів у блоках коду. Інші використовують пробіли.

Меню "Пробіли" дозволяє легко змінити тип відступів, що використовуються у поточному файлі, на тип, з яким вам зручно працювати.

Примітка: ClassBrowser працює коректно лише якщо поточний файл використовує той самий тип відступів (символ табуляції чи пробіли), що встановлено за замовчуванням для DrPython у налаштуваннях. DrPython автоматично перевірятиме це при відкритті Class Browser. Щоб виправити файл із змішаними відступами, використовуйте меню "Пробіли" у розділі "Редагування".

ПРИМІТКА: Якщо кінці рядків змішані, DrPython не зможе коректно виправити відступи. Завжди встановлюйте кінці рядків перед типом відступів.

Гарячі клавіші

Ви можете встановити гарячі клавіші, вибравши "Налаштувати гарячі клавіші" у меню Опції.

"Ignore Keys" ігнорує модифікатори, перелічені у списку. Дійсні лише модифікатори. Це в основному для операційних систем, таких як Linux, де клавіша "Meta" встановлена на "Num Lock" за замовчуванням, і тому може бути завжди увімкнена.

Примітка: Ви можете встановити гарячі клавіші для пунктів меню DrScript. Якщо ви цього не зробили, або файл гарячих клавіш якось не синхронізований із фактичною кількістю DrScripts, DrPython просто завантажить правильні DrScripts без гарячих клавіш за замовчуванням. Гарячі клавіші DrScript зберігаються окремо від гарячих клавіш DrPython.

Якщо ваші гарячі клавіші не працюють, використовуйте "get key" для гарячої клавіші, щоб перевірити, чи постійно присутній модифікатор. На Linux може бути присутня клавіша "Meta". Якщо це так, використовуйте "Ignore Keys", щоб встановити "Meta" як клавішу для ігнорування.

Для заданої гарячої клавіші просто натисніть кнопку "Get Key" і натисніть клавіші, які ви хочете. Вона покаже виявлені модифікатори та код клавіші. Коли ви закінчите, натисніть "Close", і рядок клавіш буде поруч із гарячою клавішею.

Виберіть "Оновити", щоб почати використовувати клавішу зараз. Виберіть "Зберегти", щоб DrPython запам'ятав зміни.

Примітка: На деяких платформах деякі клавіші можуть не реєструватися, і тому не можуть бути використані як гарячі клавіші.
